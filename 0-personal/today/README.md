# Daily Briefings

Generated daily briefings from `/today` and `/gtd-daily-start` commands.

## Contents

- Morning briefings with priorities
- AI work summaries
- Calendar overview
- Focus recommendations

## Usage

```
/today          # Generate today's briefing
/gtd-daily-start  # Full morning planning routine
```

## Privacy

Daily briefings are gitignored. Only this README is tracked.
